# Squirrel-Math

## Run

Install node in version declared by [.nvmrc](./App/.nvmrc). Go to [App](./App/) directory and run:

```
npm i
npm run serve
```

## Deploy

```
npm i -g firebase-tools
./deploy.ps1
```

## Redaction notes

When contributing to lesson content, follow the conventions described in [Conventions article](./App/src/components/Conventions.vue), available from Home page.
